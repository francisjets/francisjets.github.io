Proyecto 1: Se construye una pagina en html, aprendiendo la estructura mediante etiquetas

Proyecto 2: Se aprende como maquillar y dar estilo a una pagina, aprendiendo conceptos como normalize y distintas formas de ocupar css

Proyecto 3: Se aprende a ocupar flex,herramienta que nos ayuda alinear y ordenar la pagina, conceptos importantes: display flex, justify content, align items, entre otros

Proyecto 4: Se aprende a ocupar bootstrap, pagina que nos sirve como una biblioteca para poder , facilitar el proceso y agiizar la construccion de una pagina.

Proyecto 5: Se aprende a ocupar git hub , pagina que nos permite trabajar creando un repositorio , en donde podemos clonar el repositorio de otra persona , clonarlo y luego modificarlo mediante comandos como git add, git commit -m y git puch,etc
